АПОНОМЕНТ КОНФИГУРАЦИИ

Файл: landmarks_config.json

  ОПИСАНИЕ:
  Конфигурации настройк проекта и данные достопримечательностей
  
  ГЛАВНЫЕ КЛЮЧИ:
  
  - project_info: Общая информация проекта
    * name: Название проекта
    * version: Версия
    * created_date: Дата создания
  
  - map_settings: Параметры карты
    * provider: Поставщик карты (openstreetmap)
    * center: Начальная зоома и центр
    * tiles: URL для графики
  
  - landmark_categories: Набор категорий
    * id: Метрика категории
    * name: Машинное имя
    * icon_color: Цвет маркера
  
  - scientists: Выдающиеся ученые
  - landmarks: Отдельные достопримечательности

ПОНААОМЕНТ Python СКРИПТА

Файл: landmark_analyzer.py

  ПОНЕШИ:
  - ChemicalLandmark: датабертия для одной достопримечательности
  - LandmarkAnalyzer: класс и для аналитики
  
  ОСНОВНЫЕ МЕТОДЫ:
  - add_landmark(): Добавить достопримечательность
  - validate(): Проверить данные
  - get_statistics(): Получить статистику
  - export_to_json(): Экспортировать твори
  - print_report(): Вывести отчет
  
  ВЫПОЛНЕНИЕ:
  С обычным Python:
  
  $ python landmark_analyzer.py

ОНЕНЙНОЛКО:
